# VetCalc Desktop

VetCalc — 獣医臨床計算ツール (Mac/Windows デスクトップ版)

このリポジトリは [Releases](https://github.com/roon1115/vetcalc-desktop/releases) ページで DMG / exe を配布するためのものです。
ソースコードは別管理。

## ダウンロード
最新版は [Releases](https://github.com/roon1115/vetcalc-desktop/releases/latest) からどうぞ。

## 自動アップデート

VetCalc Desktop は起動時に自動で新バージョンをチェックします。
